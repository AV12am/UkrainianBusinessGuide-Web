## 🚀 Як запустити

### **Варіант 1: Прямо в браузері (найпростіше!)**

Просто перейдіть за посиланням:
```
https://av12am.github.io/UkrainianBusinessGuide-Web/
```

### **Варіант 2: Локально на комп'ютері**

1. **Клонуйте репозиторій:**
```bash
git clone https://github.com/AV12am/UkrainianBusinessGuide-Web.git
cd UkrainianBusinessGuide-Web
```

2. **Відкрийте файл в браузері:**
   - Просто двічі клікніть на `index.html`
   - Або відкрийте його через браузер (File → Open)

3. **Альтернативно, запустіть локальний сервер:**
```bash
# Якщо у вас Python 3:
python -m http.server 8000

# Потім перейдіть на:
# http://localhost:8000
```
